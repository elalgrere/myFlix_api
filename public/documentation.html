Movie API Documentation

Objective

To build an api for a movies application which provides information about films, directors, and genres. The users will be able to sing up, update their personal information, and create a list of favorite movies.

Main features

  • Allow new users to register
  • Allow users to add a movie to their list of favorites
  • Return data about all users
  • Return data about users by name
  • Return data about all movies
  • Return data about movies by title
  • Return data about a genre (description) by name/title
  • Return data about a director (bio, birth year, death year) by name
  • >Allow to remove a user by name
  • >Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister
  • Allow to delete users Favorite movie
